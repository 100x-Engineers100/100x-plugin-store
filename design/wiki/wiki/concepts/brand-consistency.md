## Definition
Every design decision reinforces the brand system rather than individual designer preference. Consistency is not sameness — it's coherence across all touchpoints.

## Why It Matters (for design)
Brand is built through repetition. Each inconsistency erodes the mental model users form of who we are. The design system exists to make consistency the path of least resistance.

## How to Apply It
- Before any color/type/spacing decision: check the system first
- If the system doesn't have an answer: document the decision, don't improvise
- AI-generated assets must pass brand filter before use (use documented prompt parameters)
- When in doubt: simpler is more consistent than clever

## When to Apply It (trigger)
Every design output before handoff. Run [[design-brief-framework]] check.
