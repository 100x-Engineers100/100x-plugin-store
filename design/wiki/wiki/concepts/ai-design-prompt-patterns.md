## Definition
Translating brand attributes into generative AI tool parameters (Midjourney, DALL-E, Stable Diffusion) so AI-generated assets are on-brand rather than generic.

## Why It Matters (for design)
AI image tools default to generic internet aesthetics. Without brand-specific prompts, outputs look off-brand even if individually good.

## How to Apply It
Base prompt template (from [[visual-identity-system]]):
```
Style: editorial photography, warm natural light, 35mm film grain
Color: warm tones, desaturated, no neon, no cold blue
Mood: calm, intentional, human
Avoid: hyperrealistic CGI, cyberpunk, neon, cold clinical lighting
```

Add project-specific subject on top. Never use without style parameters.

## When to Apply It (trigger)
Any request for AI-generated imagery for brand use.
