## Overview
Primary design tool. Source of truth for all design assets and the component library.

## Key Facts / Usage
- Design System file: source of truth for all components (link in Notion)
- This wiki doc is spec source of truth — Figma is asset source of truth
- All components must be built from Figma design system, not custom
- Icon library lives in Figma — do not create one-off icons outside it
