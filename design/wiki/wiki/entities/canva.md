## Overview
Secondary design tool for non-designer asset creation. Used when team members outside design need to create brand-consistent assets without Figma access.

## Key Facts / Usage
- Brand kit must be set up in Canva matching [[visual-identity-system]] values
- Only approved templates should be used — not blank canvases
- Outputs must be reviewed by a designer before external use
- Not for product UI design — Figma only for that

## Connections
Appears in sources: [[visual-identity-system]]

## Notes
Canva MCP is available as an optional add-on for AI-assisted design work within brand parameters.
